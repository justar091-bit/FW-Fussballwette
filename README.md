# FW-Fussballwette

Offline-Fussballwetten-Simulator mit virtuellem Spielgeld.

## Starten
Öffne `index.html` im Browser.

## GitHub Pages
1. Dateien in dein Repository hochladen.
2. Settings → Pages.
3. Deploy from a branch auswählen.
4. Branch `main` und `/root` auswählen.
5. Save drücken.

Nur Spielgeld. Kein echtes Geld.
